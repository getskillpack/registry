# Hello skill

Этот пакет — **образец структуры** для публикации в [registry](https://github.com/getskillpack/registry). В архиве должен быть хотя бы один файл скилла (например `SKILL.md` в корне tarball).

## Когда использовать

- Проверка `skpkg` / клиента реестра.
- Шаблон для собственных скиллов.

## Содержимое

Один файл `SKILL.md` с YAML frontmatter (`name`, `description`).
